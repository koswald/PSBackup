SHORT DESCRIPTION

Provides customizable backups.

LONG DESCRIPTION

Provides a PowerShell script and supporting modules and documentation to perform file and folder backups.

EXAMPLES

./Setup.ps1 # Setup the project for the first time
$specs = @(@{
    Name = "My PowerShell scripts"
    Src = "$home/PowerShell"
    Dest = "$home/Backups/MyPSScripts"
    Include = "*.ps*", "*.md", "*.txt"
    Subfolders = $true
})
Backup-Updates $specs
